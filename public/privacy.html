🏗 How Bulbring AI Works

Bulbring AI is a private web application hosted on Render (a secure cloud platform). When you log in and use any of the tools, here is exactly what happens:

  • Login: You enter your username and password. Your password is never stored in plain text — it is hashed with bcrypt (a one-way encryption algorithm). Even the site operator cannot read your password. After your password is verified, you enter a private PIN. A short-lived session token (JWT) is issued and stored only in your browser's local storage.
  • Bank Statement Analyzer & Audit Risk Scorer: You upload a PDF. The file is loaded into server memory, text is extracted, and the text is sent to the Claude AI API. The PDF is never written to disk and is cleared from memory as soon as the response is returned. The resulting Excel file is sent directly to your browser.
  • Receipt Scanner: You upload an image (JPEG, PNG, WebP). The image is sent to Claude AI Vision as a base64-encoded payload to extract line items. The image is not stored anywhere.
  • AI Accountant Chat: Your typed questions are sent to Claude AI and a response is returned. Messages are held temporarily in your browser's memory for the length of your conversation and are cleared when you close the page or click "Clear".
  • P&L, HST, Invoice, Trial Balance: You enter numbers directly. These are processed server-side and an Excel file is generated and returned. No input data is retained.
  • File History: Generated files are held in the server's working memory for the duration of the current server session so you can re-download them. This list resets every time the server restarts and is never written to a database or disk.

📤 What Data Leaves This Server

Some data is sent to a third-party AI provider to power the AI features:

Anthropic (Claude AI API): When you use any AI-powered feature, the relevant text or image is transmitted to Anthropic's API over an encrypted (HTTPS/TLS) connection. Anthropic's API processes your request and returns a result. Anthropic's data retention policies apply — by default, API inputs and outputs are not used to train models and are subject to Anthropic's enterprise privacy terms. You can review Anthropic's privacy policy at anthropic.com/privacy.

No other third parties receive your data. There are no analytics services, advertising networks, or data brokers involved.

🗑 What Is Never Stored

  • Your uploaded PDF bank statements or receipt images are never saved to disk.
  • Transaction data extracted from your statements is never written to a database.
  • Generated Excel files are held only in server memory and cleared on restart.
  • Your chat messages with the AI Accountant are not logged or stored.
  • No cookies are used. Your session is stored only in your browser's local storage.
  • No IP addresses, usage logs, or browsing behaviour are tracked.

🔐 Security Measures in Place

  • HTTPS / TLS: All communication between your browser and this server is encrypted in transit. Any HTTP request is automatically redirected to HTTPS.
  • Password hashing: Your login password is hashed with bcrypt (cost factor 12) before storage. It cannot be reversed or read.
  • Two-step login: After entering your password, a second private PIN is required. This prevents access even if your password were ever compromised.
  • JWT session tokens: Your session is represented by a signed JSON Web Token stored in browser local storage. Tokens expire automatically and are validated on every request.
  • Rate limiting: All API routes are rate-limited to block brute-force attacks and prevent abuse. AI-powered routes have a tighter hourly limit.
  • HTTP security headers: The server sets Helmet.js security headers including Content Security Policy, X-Frame-Options, X-Content-Type-Options, and Referrer-Policy to mitigate common web attacks.
  • File type validation: Uploads are restricted to specific file types (PDF for statements, JPEG/PNG/WebP for receipts) and size limits. Invalid files are rejected before processing.
  • Formula injection prevention: All user-supplied text written into Excel files is sanitized to prevent formula injection attacks.
  • Single-user access: This application is configured for a single authorized user. There are no public registration endpoints.

🌐 Where Your Data Is Hosted

Bulbring AI is hosted on Render (render.com), a cloud hosting platform based in the United States with SOC 2 Type II certification. All server infrastructure uses encrypted storage and secure network configurations.

Note for Canadian users: While this application is built for Canadian tax purposes (CRA T2125, HST/GST), the server infrastructure is located in the United States. Data transmitted to the server (including PDF contents) crosses international borders. If you have strict data residency requirements, please keep this in mind.

✅ Your Rights & Controls

  • Delete your session at any time: Click "Sign Out" to immediately invalidate your session token.
  • File history: The in-session file history resets automatically every time the server restarts. No permanent record is kept.
  • No account to delete: Since this is a single-user private tool with no database of user accounts, there is no account profile to delete.
  • Questions: For any privacy concerns, contact the site owner directly.

🌿 The Short Version

Your financial data is used only to generate the output you asked for. It is processed in memory, sent to Claude AI to power AI features, and then discarded. Nothing is sold, nothing is tracked, and nothing is permanently stored on this server. This is a private tool built for your use only.

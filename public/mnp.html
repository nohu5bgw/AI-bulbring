Bank Writeup Agent

Drop in an RBC Visa statement (PDF or Excel). The agent uploads it securely to the Bulbring AI backend, where Claude categorizes every transaction to CRA expense codes, and exports a fully formatted MNP-branded writeup Excel back to your machine.

Windows 10 / 11 (.exe)  ·  macOS 12+ (.zip with .app)  ·  no install needed
How the agent works
1

Drop in a statement

Browse to your RBC Visa PDF or Excel export. The agent reads every line — dates, descriptions, foreign currency, exchange rates, cardholder names.

2

Auto-categorize

Each transaction is matched to a CRA-style category (meals, office, vehicle, travel, etc.). Anything uncertain is flagged orange so you can review it.

3

Review & edit inline

Click any description or category to edit. Subtotals update on the fly. Payments are colour-coded green for quick visual reconciliation.

4

Export the writeup

One click generates a two-sheet Excel: full detail with cardholder subtotals plus a one-page category Summary — ready for the working paper file.

What you get

Two-sheet Excel

Detailed writeup + category summary, MNP-branded.

Cardholder subtotals

Primary & supplementary cards grouped automatically.

FX handling

Foreign currency, exchange rate & CAD preserved.

Mismatch warnings

Flags if parsed totals don’t reconcile to the statement.

Same AI as the web tool

Uses the Bulbring backend — same prompt, same Excel output.

Plain .xlsx

No macros. Opens in any version of Excel.

Install & first run — Windows
  1. Click Download for Windows. Your browser saves MNP_BankWriteup.exe — usually to your Downloads folder.
  2. Drag it to your Desktop (or wherever you want it to live).
  3. Double-click the icon to launch the app. No install, no setup, no Python required.
  4. The first time you click Generate Writeup, the app asks for your Bulbring AI PIN. Enter it once — it’s saved locally so you won’t be asked again.

First launch may take a few seconds while Windows unpacks the bundled libraries. If SmartScreen warns about an unrecognized publisher, click More info → Run anyway — the app is unsigned because it’s an internal build.

Install & first run — macOS
  1. Click Download for Mac. Your browser saves MNP_BankWriteup_Mac.zip.
  2. Double-click the zip to extract it. You’ll get MNP_BankWriteup.app. Drag it to your Applications folder (or Desktop).
  3. First launch only: right-click the app → Open → click Open in the dialog. After that you can double-click normally.
  4. The first time you click Generate Writeup, the app asks for your Bulbring AI PIN. Enter it once — it’s saved locally so you won’t be asked again.

macOS Gatekeeper blocks unsigned apps by default. The right-click → Open dance is a one-time bypass — needed because the build isn’t signed with an Apple Developer ID. If you see “cannot be opened because Apple cannot check it for malicious software”, go to System Settings → Privacy & Security and click Open Anyway.

Bank Writeup Agent

Drop in an RBC Visa statement (PDF or Excel). The agent parses every transaction, auto-categorizes to CRA expense codes, and exports a fully formatted MNP-branded writeup Excel — all running locally on your computer.

⬇   Download for Windows .zip · 33 KB · Windows 10 / 11
How the agent works
1

Drop in a statement

Browse to your RBC Visa PDF or Excel export. The agent reads every line — dates, descriptions, foreign currency, exchange rates, cardholder names.

2

Auto-categorize

Each transaction is matched to a CRA-style category (meals, office, vehicle, travel, etc.). Anything uncertain is flagged orange so you can review it.

3

Review & edit inline

Click any description or category to edit. Subtotals update on the fly. Payments are colour-coded green for quick visual reconciliation.

4

Export the writeup

One click generates a two-sheet Excel: full detail with cardholder subtotals plus a one-page category Summary — ready for the working paper file.

What you get

Two-sheet Excel

Detailed writeup + category summary, MNP-branded.

Cardholder subtotals

Primary & supplementary cards grouped automatically.

FX handling

Foreign currency, exchange rate & CAD preserved.

Mismatch warnings

Flags if parsed totals don’t reconcile to the statement.

Offline & private

Runs locally. No client data leaves the machine.

Plain .xlsx

No macros. Opens in any version of Excel.

Install & first run
  1. Right-click the downloaded zip and choose Extract All.
  2. Open the extracted folder and double-click Run.bat.
  3. If Python isn’t installed, the launcher opens the Python download page. Install Python 3.11+ and tick “Add Python to PATH” during install, then run Run.bat again.
  4. First run takes about a minute (installing the libraries). Subsequent launches open instantly.

If Windows SmartScreen warns about Run.bat, click More info → Run anyway. It’s a plain text launcher you can open in Notepad to inspect.

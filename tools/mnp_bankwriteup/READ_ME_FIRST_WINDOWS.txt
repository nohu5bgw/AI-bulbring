MNP Bank Writeup — Windows quick start
=======================================

1. Unzip this folder anywhere on your computer.

2. Double-click "Run.bat".

   - If Python is not installed, the script will open the Python download
     page. Install Python 3.11 or newer. IMPORTANT: during install, tick
     the box "Add Python to PATH". Then double-click Run.bat again.

   - First run takes ~1 minute while it installs the libraries it needs.
     Subsequent runs start immediately.

3. The app window opens. Use it as normal:
   - Click "Browse PDF / XLSX" and pick an RBC Visa statement.
   - Review the categorized transactions.
   - Click "Generate Writeup Excel" to save the output.

If Windows SmartScreen warns about Run.bat, click "More info" then
"Run anyway" — it's a plain text launcher script, you can open it in
Notepad to inspect.
